📦 Export & Import Data

Export saves all your food logs and weight history as a file on your phone. Import restores it anytime — useful if you change phones or clear browser data.

💡 Export your data once a week as a habit. Save the file to Google Drive or WhatsApp it to yourself for safekeeping.
📊 Google Sheets Auto-Sync

Every time you log food, it automatically saves a copy to your personal Google Sheet. Works as a cloud backup and lets you view your data on any device.

Your Google Apps Script URL
How to set up Google Sheets sync
Step 1. Go to sheets.google.com → Create a new blank spreadsheet
Step 2. Click Extensions → Apps Script
Step 3. Delete everything in the editor and paste the script below
Step 4. Click 💾 Save → Click ▶ Run → Allow permissions
Step 5. Click Deploy → New deployment → Type: Web App
Step 6. Set "Who has access" to Anyone → Click Deploy
Step 7. Copy the Web App URL → Paste it above 
function doPost(e) {
  try {
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    var data = JSON.parse(e.postData.contents);
    var timestamp = new Date().toLocaleString('en-IN');
    sheet.appendRow([
      timestamp,
      data.date,
      data.totalCal,
      data.burned,
      data.steps,
      data.water,
      data.weight || '',
      data.meals
    ]);
    return ContentService.createTextOutput(JSON.stringify({status:'ok'}))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService.createTextOutput(JSON.stringify({status:'error',msg:err.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet(e) {
  return ContentService.createTextOutput('Health Tracker Sync Active ✅')
    .setMimeType(ContentService.MimeType.TEXT);
}
